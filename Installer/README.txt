# Petty Cash Management System
## Ceylon Electricity Board - Haliela

### Welcome!

Thank you for choosing the Petty Cash Management System. This software will help you manage petty cash expenses efficiently and generate official CEB reports.

### What will be installed:

1. **Petty Cash Desktop Application**
   - Modern Windows application for managing expenses
   - Generate official CEB reports in Sinhala
   - User management and audit logging

2. **SQL Server Express LocalDB**
   - Lightweight database engine
   - No service required - runs on-demand
   - Perfect for single-user desktop applications

3. **Sinhala Fonts**
   - Iskoola Pota font for proper Sinhala text display
   - Automatically installed to Windows Fonts folder

4. **Automated Database Setup**
   - Database and tables created automatically
   - Default admin user and categories pre-configured

### System Requirements:

- Windows 10 or Windows 11 (64-bit)
- .NET 8.0 Runtime (will prompt to download if not installed)
- 500 MB free disk space
- Administrator privileges (for installation only)

### Installation Time:

- Approximately 5-10 minutes depending on your system
- SQL Server installation may take a few minutes

### After Installation:

**Default Login Credentials:**
- Username: `admin`
- Password: `admin123`

**IMPORTANT:** Please change the admin password immediately after first login!

### Need Help?

Contact your system administrator or IT support team.
